## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

To run locally yoou need mongodb database

```
docker-compose up -d
```

- MongoDB URL Local:

```
  mongodb://localhost:27017/entriesdb
```
